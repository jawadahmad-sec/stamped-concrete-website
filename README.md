# Decorative Concrete Flooring – Stamped Concrete Website

Bilingual (Arabic/English) business website for Zia Stamped Concrete Flooring, Saudi Arabia.

## How to update the gallery
Add new images to `images/gallery/` and update `js/gallery.js` with the new filenames and captions.

## Contact form
The contact form is a Netlify form. Submissions are forwarded to the client's email automatically.

## Deployment
Deployed on Netlify. Automatic deployments from the `main` branch of this repository.
